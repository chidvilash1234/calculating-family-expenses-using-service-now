# Calculating Family Expenses using ServiceNow

## Project Overview
This project is an individual academic project developed using the ServiceNow platform.  
The main objective of this project is to calculate, manage, and analyze family expenses in a centralized system.

Manual expense tracking using notebooks or spreadsheets is time-consuming and error-prone.  
This project replaces manual methods with a structured ServiceNow-based solution.

## Project Type
- ServiceNow Administrator Project
- Configuration-based (No coding)

## Objectives
- Track daily and monthly family expenses
- Categorize expenses (Food, Rent, Bills, etc.)
- Reduce errors in manual calculations
- Generate expense reports for better financial planning
- Use a centralized and secure platform
